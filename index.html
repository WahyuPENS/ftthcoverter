📋 Format yang Didukung

Input: File KML (.kml) — export dari Google Earth atau QGIS


Output DXF: AutoCAD 2010+ (.dxf) — kompatibel dengan AutoCAD, BricsCAD, ZWCAD


Output CSV: UTF-8 CSV — kolom: Name, Latitude, Longitude, Elevation, Description


Ukuran file maksimal: 50MB per upload.

🗺️ Cara Penggunaan

1. Pilih fungsi yang dibutuhkan (KML ke DXF atau KML ke CSV)


2. Upload file KML dengan klik area upload atau drag & drop


3. Klik tombol Konversi dan tunggu proses selesai


4. File hasil konversi akan otomatis terunduh ke komputer Anda


Pastikan file KML memiliki koordinat yang valid.

🔧 Troubleshooting & FAQ

File tidak bisa diupload?
Pastikan ekstensi .kml dan ukuran di bawah 50MB.


Konversi gagal?
File KML mungkin format tidak standar. Coba export ulang dari QGIS atau Google Earth.


Koordinat tidak muncul di CSV?
Periksa apakah KML memiliki elemen <coordinates> yang valid.
